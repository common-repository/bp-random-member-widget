=== BP Random Member Widget ===
Contributors: tom@yaway
Tags: Buddypress, Member, Widget
Requires at least: WP 2.9.1, BuddyPress 1.2
Tested up to: WP 2.9.2, BuddyPress 1.2
Stable tag: 1.2

This Plugin adds a sidebar widget with avatar and username of a random BuddyPress user.

== Description ==

English: This Plugin adds a sidebar widget with avatar and username of a random BuddyPress user.

German: Dieses BuddyPress-Plugin erstellt ein Sidebar-Widget, welches ein zuf&auml;lliges Mitglied mit Avatar und Namen anzeigt, ohne den ganzen anderen Schnick-Schnack des Standard-Widgets.


== Installation ==
= English Version: =
1. Upload this plugin to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the BP-RandomMemberWidget to your sidebar.
4. Done!

= German Version: =
1. Pluginordner in das Verzeichnis `/wp-content/plugins/` hochladen.
2. Plugin via Adminpanel aktivieren.
3. Das BP-RandomMemberWidget zum Sidebar hinzuf&uuml;gen.
4. Fertig!
